# AI Programming – Data Science Salary Analysis 2024

## Project Overview
This is an academic project for **Artificial Intelligence Programming** that analyzes **2024 data science salaries** using Python.
The project demonstrates a complete workflow: **data loading → cleaning → preprocessing → analysis → visualization**.

---

## Dataset
- **File:** `DataScience_salaries_2024.csv`
- **Contains:** salary in USD, experience level, job title, work year, and other job-related fields.

---

## Tools & Libraries
- Python
- Pandas
- Scikit-learn (StandardScaler)
- Matplotlib
- Seaborn
- Statistics

---

## Methodology
1. Load and explore the dataset (sample + info)
2. Clean data (convert salary to numeric, remove missing values, remove duplicates)
3. Preprocess features:
   - Scale salary using **StandardScaler**
   - Encode experience level (EN, MI, SE, EX) into numeric values
   - Group job titles into categories (e.g., Data Science, AI & ML, Management…)
4. Perform analysis:
   - Mean salary per job category
   - Correlation between salary and experience level
5. Visualize results:
   - Line plot (average salary per job category)
   - Bar chart (employees per category)
   - Heatmap (correlation)
   - Histogram (salary distribution)
   - Scatter plot (salary vs experience)
